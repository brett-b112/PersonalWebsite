Kansas City Barbeque Society Data Analysis
Growing up my dad would compete in the American Royal and each october I would make my way downtown to enjoy all the barbeque that KC has to offer.

That tradition inspired me to make this project. I wondered if there was any data on these competitions and what I could do. I was able to find a dataset on kaggle and able to view some of my childhood experiences with concrete data.

With this dataset I was able to create a heatmap of competitions per state, the average competition location based on longitude and latitude, and some valuable insights on competiton scores by meat.
